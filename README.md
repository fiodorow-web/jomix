# Jomix — Sklep internetowy

Next.js 16 storefront dla marki odzieżowej [Jomix](https://jomix.pl).

## Stack

- **Framework:** Next.js 16 (App Router, Turbopack)
- **Styling:** Tailwind CSS 4
- **Hosting:** Vercel
- **Commerce:** Shopify (Storefront API — w przygotowaniu)
- **Płatności:** Przelewy24
- **Wysyłka:** InPost ShipX

## Uruchomienie lokalne

```bash
npm install
npm run dev
```

Strona dostępna na [http://localhost:3000](http://localhost:3000).
